Configuration

Discovery uses the free OpenStreetMap Overpass API (no key). Auditing real cross-origin sites needs a fetch proxy — browsers block direct reads. Pick one:

Free options that work out of the box: https://r.jina.ai/ (prepends, returns readable text+headers) or https://corsproxy.io/?url=. For production reliability, deploy the included Cloudflare Worker (in the README) and paste its URL. Leave blank to run in simulation mode (synthetic audits for demos).

AI review & outreach run through the host's Claude API — no key needed inside this environment. On your own deploy, route those calls through a serverless function with your Anthropic key.
